# 小程序日历

### 思路分析

要实现一个简单的日历，需要先知道几个值：

- 每月有多少天

- 每月第一天、最后一天各星期几

- 每月1号之前应有多少天属于上一个月

- 每月最后一天之后应有多少天属于下一个月

> 每月最多31天，最少28天。

### 引入 wxml 及 wxss后初始化日历组件直接使用
```js
import initCalendar from '../../template/calendar/index';
const conf = {
  onShow() {
    initCalendar(); // 使用默认配置初始化日历
  }
};
Page(conf);
```

更多配置及功能请 [参考文档](http://isfeer.com)
